  • Craft
    웹 퍼블리싱에서 가장 중요한 것은 기본기를 바탕으로 세심하게 작업하는 것입니다. HTML, CSS를 활용하여 웹 페이지의 구조와 스타일을 정확하게 구현하고, 사용자가 편리하게 이용할 수 있도록 고려했습니다. 각 요소의 디테일까지 신경 써서 완성도를 높이며, 항상 최적의 결과를 목표로 작업했습니다.
  • Learn
    GSAP, SCSS 등 다양한 웹 기술을 활용해 웹 퍼블리싱 역량을 확장했습니다. JavaScript , Jquery 더해 애니메이션과 스타일링을 최적화하는 기술을 적용했습니다. 배운 기술을 바로 적용하며, 점점 더 효율적인 방법을 찾고 있습니다.
  • Optimize
    웹 퍼블리싱 과정에서 발생한 문제들을 분석하고 효율적인 방법으로 해결했습니다. 반복되는 코드나 불필요한 자원을 최적화하여 페이지 성능을 향상시켰습니다. 효율적이고 관리가 쉬운 코드를 작성하여 웹 사이트의 품질을 높였습니다.
